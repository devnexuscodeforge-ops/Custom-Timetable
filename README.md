# Custom-Timetable
# 🚀 Custom Timetable Generator

A modern and interactive timetable generator built with **HTML, CSS, and JavaScript** that allows users to create their own personalized study schedules.

Users can:
- Add subjects
- Set study times
- Track study hours
- Delete tasks
- Create fully custom timetables

---

# 🌟 Features

- 🎨 Modern Dark UI Design
- 📚 Custom Subject Input
- ⏰ Start & End Time Selection
- 🧮 Total Study Hours Counter
- ➕ Add Timetable Cards
- 🗑 Delete Study Sessions
- 📱 Responsive Design
- ⚡ Fast and Lightweight
- 💻 Pure HTML, CSS & JavaScript
